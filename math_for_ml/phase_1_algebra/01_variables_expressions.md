## Practice Working

### Q1 — Substitution (c = 60)
P = 5c - 200
P = 5 * 60 - 200
P = 300 - 200
P = 100
Result: Profit of ₹100

### Q2 — Solve for break-even (P = 0)
Formula: P = 8c - 400
Step 1: Substitute P = 0
0 = 8c - 400
Step 2: Add 400 to both sides
400 = 8c
Step 3: Divide both sides by 8
c = 400 / 8
c = 50
Result: Need to sell 50 cups to break even

### Q3 — ML formula thinking
y = wx + b

| Symbol | Type        | Meaning                        |
|--------|-------------|--------------------------------|
| x      | Variable    | Input — changes every time     |
| w      | Coefficient | Multiplier attached to x       |
| b      | Constant    | Fixed value, always present    |

Real life example — Food budget:
x = number of times eating outside
w = cost per meal (60$)
b = fixed food cost (groceries etc)
y = total food spend

y = 60x + b
If x = 5: y = 60*5 + b = 300 + b

## My Understanding

Variable    : Something that keeps changing, we can't predict its value

Constant    : Always fixed before the equation starts, never changes

Coefficient : The multiplier attached to a variable
